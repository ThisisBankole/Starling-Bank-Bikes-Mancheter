# Cycle Tracker API

Live bike-share availability built on public GBFS feeds. Data refreshes every 10 minutes and includes 90 days of history.

Free to use. No API key required.

```
https://cycle.arrakis.house/api/v1
```

| City | id |
| --- | --- |
| Manchester | `manchester` |
| Bournemouth, Christchurch & Poole | `bcp` |
| Bradford | `bradford` |
| Brighton & Hove | `brighton` |

There are two ways to consume the data: connect an AI agent via **MCP**, or call the **REST API** directly.
