# Cycle Tracker API

Live bike-share availability for **Manchester**, **Bournemouth, Christchurch & Poole**, and **Bradford**, built on Beryl's public GBFS feeds. Data refreshes every 10 minutes and includes 90 days of history.

Free to use. No API key required.

```
https://cycle.arrakis.house/api/v1
```

| City | id | Stations |
| --- | --- | --- |
| Manchester | `manchester` | ~286 |
| Bournemouth, Christchurch & Poole | `bcp` | ~400 |
| Bradford | `bradford` | ~33 |

There are two ways to consume the data: connect an AI agent via **MCP**, or call the **REST API** directly.
